**backend/app/sources/service.py** -- Create the ingestion service module:

1. **Docling converter singleton** (per RESEARCH.md Code Examples):
   ```python
   from functools import lru_cache
   from docling.document_converter import DocumentConverter, PdfFormatOption
   from docling.datamodel.base_models import InputFormat
   from docling.datamodel.pipeline_options import PdfPipelineOptions

   @lru_cache
   def get_converter() -> DocumentConverter:
       pipeline_options = PdfPipelineOptions()
       pipeline_options.do_ocr = False  # D-06
       pipeline_options.do_table_structure = True
       pipeline_options.do_code_enrichment = False
       pipeline_options.do_formula_enrichment = False
       pipeline_options.document_timeout = 300.0
       return DocumentConverter(
           allowed_formats=[InputFormat.PDF, InputFormat.HTML],
           format_options={InputFormat.PDF: PdfFormatOption(pipeline_options=pipeline_options)},
       )
   ```

2. **`async def parse_pdf(file_path: Path, filename: str, session: Session) -> AsyncGenerator`** -- yields `IngestionProgress` and `IngestionComplete`:
   - Yield progress: `status="parsing", message="Parsing document..."`
   - Run docling in threadpool: `result = await asyncio.to_thread(get_converter().convert, str(file_path))` (Pitfall 1: CPU-bound, must not block event loop)
   - Extract markdown: `markdown = result.document.export_to_markdown()`
   - Get page count: `page_count = len(result.document.pages)`
   - D-01 scanned detection: `avg_chars = len(markdown) / max(page_count, 1)` -- if `avg_chars < 50`, set `parse_status = "warning"` and add `"scanned_pdf"` to warnings list
   - D-02 duplicate check: `content_hash = hashlib.sha256(markdown.encode()).hexdigest()` -- query `session.exec(select(RawSource).where(RawSource.content_hash == content_hash)).first()`. If exists, yield error event with existing source ID and return.
   - D-04 extract metadata: Try to get title from `result.document.name` or `result.document.origin.filename`. Fall back to filename without extension.
   - Save raw PDF to disk: `{settings.clinic_atlas_sources_dir}/{content_hash}.pdf` (path traversal safe -- hash-based filename, per T-02-01)
   - Create `RawSource(content=markdown, content_hash=content_hash, file_path=disk_path, title=title, mime_type="application/pdf", parse_status=parse_status, page_count=page_count, source_type="pdf")`
   - `session.add(source)`, `session.commit()`, `session.refresh(source)`
   - Yield progress: `status="indexing", message="Indexing..."`
   - Yield complete: `IngestionComplete(id=source.id, title=source.title, parse_status=source.parse_status, page_count=source.page_count, content_preview=markdown[:500], source_type="pdf", warnings=warnings)`

3. **`async def fetch_and_parse_url(url: str, session: Session) -> AsyncGenerator`** -- yields `IngestionProgress` and `IngestionComplete`:
   - Yield progress: `status="fetching", message="Fetching..."`
   - Fetch with httpx: `async with httpx.AsyncClient(timeout=settings.httpx_timeout, follow_redirects=True) as client: resp = await client.get(url)`. Raise on 4xx/5xx.
   - SSRF mitigation (T-02-03): After following redirects, check `resp.url` -- reject if final host resolves to private IP ranges (10.x, 172.16-31.x, 192.168.x, 127.x, ::1). Use `ipaddress.ip_address(socket.gethostbyname(resp.url.host))` and check `.is_private`.
   - Yield progress: `status="extracting", message="Extracting content..."`
   - Parse HTML with docling in threadpool: `result = await asyncio.to_thread(get_converter().convert_string, content=resp.text, format=InputFormat.HTML, name="page")`
   - `markdown = result.document.export_to_markdown(strict_text=True)`
   - D-07 thin content check: if `len(markdown) < 200`, add `"thin_content"` to warnings
   - D-08 JS fallback: if thin content, yield progress `status="extracting", message="Retrying with browser rendering..."`, then:
     ```python
     from playwright.async_api import async_playwright
     async with async_playwright() as p:
         browser = await p.chromium.launch(headless=True)
         page = await browser.new_page()
         await page.goto(url, wait_until="networkidle", timeout=settings.playwright_timeout)
         html = await page.content()
         await browser.close()
     ```
     Re-parse with docling. If still thin, keep the warning.
   - D-09 title: Extract from `result.document.name` or fall back to URL domain+path
   - D-02 duplicate check (same as PDF)
   - Create RawSource with `source_type="url"`, `url=url`, `mime_type="text/html"`
   - Yield complete event

4. **`def validate_upload_size(content_length: int, max_mb: int) -> None`** -- raises `HTTPException(413)` if over limit (D-03)

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client -> upload endpoint | Untrusted file uploads from browser |
| backend -> external URLs | SSRF risk when fetching user-provided URLs |
| backend -> SearXNG | Internal service, trusted |
| backend -> filesystem | Storing files to disk |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | POST /upload | mitigate | Validate MIME type (application/pdf), file extension (.pdf), enforce 50 MB limit. Hash-based filename prevents path traversal. |
| T-02-02 | Tampering | Stored markdown | accept | react-markdown sanitizes output. Docling produces markdown, not raw HTML. Single-user app, low risk. |
| T-02-03 | Spoofing | POST /fetch | mitigate | Validate URL scheme (http/https only). After redirect resolution, check final host is not private IP (10.x, 172.16-31.x, 192.168.x, 127.x). Reject SSRF attempts. |
| T-02-04 | Denial of Service | POST /upload | mitigate | 50 MB file size limit (D-03). Docling document_timeout=300s prevents infinite parsing. |
| T-02-05 | Information Disclosure | POST /fetch | accept | URL content is fetched and stored. Single-user app -- the user controls what URLs they submit. No multi-tenant data leak risk. |
</threat_model>
